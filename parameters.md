# 参数规定

> 编码统一使用大写

### state - 通讯状态

| **编码** | **描述** |
| :--- | :--- |
| SUCCESS | 成功 |
| FAIL | 失败 |
| VER | 版本过低 |

### code - 状态码

| **编码** | **描述** |
| :--- | :--- |
| 10000 | 成功 |
| 10001 | 用户支付中 |
| 10002 | 签名错误 |
| 10003 | 收单机构响应结果签名错误 |
| 10004 | 参数错误 |
| 10005 | 商户错误 |
| 10006 | 系统错误 |
| 10007 | 支付错误 |

### trade\_state - 交易状态

| **编码** | **描述** |
| :--- | :--- |
| SUCCESS | 交易成功 |
| FAIL | 交易失败 |
| PROCESSING | 交易处理中 |

### paytype - 支付方式

| **编码** | **描述** |
| :--- | :--- |
| WECHAT | 微信支付 |
| ALIPAY | 支付宝支付 |
| JDPAY | 京东支付 |
| QQ | QQ钱包支付 |
| UNIONPAY | 银联支付 |
| BESTPAY | 翼支付 |
